##  About

**CP Tracker** is an Android app built using **Jetpack Compose** and **Material3** that enables you to monitor your CP stats in real time. Currently, it integrates with:
- **Codeforces API**
- **LeetCode (GraphQL)**

##  Tech Stack

| Layer       | Technologies Used                                    |

| Architecture| MVVM                                                |
